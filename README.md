# MainBoard

스프링을 이용한 게시판 만들기

• 개발환경<br>
DB: Oracle<br>
Server: Tomcat 9.0<br>
IDE: Eclipse<br>
Framework: Spring, MyBatis<br>
사용언어 및 라이브러리: Java JSP Maven JSTL jQuery<br>
